# Room Management

**Description**: Creates and Updates a User in the system

**Path**: `/rooms` 

**Type**: `HTTP`

# Create Room

**Description**: Creates a Room of `RoomTypeID`, Initializes it with a User with `UserID` and Returns `RoomID`.

**Path**: `/`

**Method**: `POST`

**Inputs**:

- UserID
- RoomTypeID

**Outputs**:

- RoomID
