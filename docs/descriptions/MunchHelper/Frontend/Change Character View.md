# Change Character View

Change a character view:

1. Change name
2. Change gender
3. Change color
4. Change, add and remove race
5. Change, add and remove class
6. Increase/decrease level (minimum: 1 lvl)
7. Increase/decrease strength (strength can go below 0)
8. Kill the character
9. Save/discard changes
10. Go back (discards changes)
